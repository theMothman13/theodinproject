HTML Practice

There are a few different ways to modify text using base HTML. Things such as strong will make text important and print it in bold like such: This text is important!

We can also use em (short for emphasis), to emphasize text and make it italic. For example: this is more important than that.

To keep track of what your code is doing without displaying it to the user, you'll want to use comments. Comments are created by inserting "" into your code. Check the source code for this page for an example.
